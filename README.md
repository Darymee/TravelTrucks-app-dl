# TravelTrucks

## Опис
TravelTrucks — вебзастосунок для перегляду та бронювання кемперів. Дозволяє
знаходити кемпери у каталозі, переглядати деталі, фото та відгуки, а також
надсилати заявку на бронювання.

## Функціональність
- каталог кемперів з пагінацією (load more)
- фільтрація за локацією, типом та обладнанням
- сторінка деталей кемпера з галереєю, характеристиками та відгуками
- форма бронювання з валідацією полів

## Вимоги
- Node.js (LTS рекомендовано)
- npm

## Встановлення
```bash
npm install
```

## Запуск у режимі розробки
```bash
npm run dev
```

## Збірка проєкту
```bash
npm run build
```

## Перегляд зібраної версії
```bash
npm run preview
```

## Перевірка стилю коду
```bash
npm run lint
```
